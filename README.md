# GeoBird

A simple dashboard made to display a list of recordings of birds extracted from [Xeno Canto](https://xeno-canto.org/explore/api).

## Available Scripts

In the project directory, you can run:

### `npm start`

### `npm test`

### `npm run build`

### `npm run eject`
